простой метроном, сделанный на jQuery 

фишечки:
1) регулировка BPM от 40 до 200 ударов
2) звуковые щелчки реализованы через Web Audio API
3) кнопочка "наиграть темп" для определения темпа по кликам пользователя
4) фича сохранения настроек метронома при перезагрузке страницы
5) есть только два самых популярных музыкальных размера, это 3/4 и 4/4

используемые технологии:
HTML5 - наполнение
CSS3 - стилек
jQuery - интерактивные элементы
Web Audio API - генерация звуков
LocalStorage - сохранение состояния
